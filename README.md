## 🤔 **Quel est ce projet?**

***Ce projet à été réalisé dans le cadre de ma 2ème année du BTS. C'est un chat de discussion en temps réel en NODE.JS***


**Fonctionnalités :**

* Un Channel public,* 
* Un système de name,*
* Un Channel privés pour les messages entre deux utilisateurs.*

---

## ⚡ **Installation**

**Etape**
Pour obtenir une copie opérationnelle, suivez ces étapes d'exemple simples.

Prérequis
Installer la dernière version de npm si ce n'est pas déjà fait :


1. *Installer la dernière version de npm*
    ```
    npm install npm@latest -g
    ```
2. *Mettre en place* le git clone
   ```git clone https://github.com/TheRyshun/SioTchat.git
   ```
3. *Paramétrer* l'informations du port dans server.js
    ```
    const PORT = 'ENTREZ VOTRE PORT';
    ```

⚠️ *Attention, certains réglages sont obligatoires*⚠️
